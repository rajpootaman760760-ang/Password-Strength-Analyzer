# Password Strength Analyzer 🔐

A Python-based password security analyzer.

## Features

- Password strength scoring (0-100)
- Checks uppercase, lowercase, digits and special characters
- Detects common passwords
- Detects keyboard patterns
- Detects repeated characters
- SHA-256 password hashing demonstration
- Prevents password reuse using hash storage
- Secure password generator using secrets module

## Technologies

- Python
- hashlib
- regex
- JSON

```bash
python password_strength_analyzer.py
